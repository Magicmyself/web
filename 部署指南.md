# 🚀 网站部署指南

## 方案一：免费静态托管（推荐）

### 1. GitHub Pages（完全免费）

**步骤：**
1. 注册 GitHub 账号：https://github.com
2. 创建新仓库，命名为 `mikun-vision`（或您的用户名.github.io）
3. 上传项目代码到仓库
4. 进入仓库 Settings → Pages
5. 选择 Source 为 "Deploy from a branch"
6. 选择 Branch 为 "main"，文件夹为 "/ (root)"
7. 点击 Save，等待几分钟
8. 访问地址：`https://您的用户名.github.io/mikun-vision`

**优点：**
- 完全免费
- 自动 HTTPS
- 全球 CDN 加速
- 与 Git 版本控制集成

---

### 2. Vercel（推荐，免费且简单）

**步骤：**
1. 注册 Vercel 账号：https://vercel.com（可用 GitHub 账号登录）
2. 点击 "Add New Project"
3. 导入您的 GitHub 仓库
4. 框架预设选择 "Vite"
5. 点击 Deploy
6. 等待部署完成，获得免费域名

**优点：**
- 完全免费
- 自动 HTTPS
- 全球 CDN
- 每次推送自动重新部署
- 支持自定义域名

---

### 3. Netlify（免费）

**步骤：**
1. 注册 Netlify 账号：https://www.netlify.com
2. 点击 "Add new site" → "Import an existing project"
3. 连接 GitHub 账号
4. 选择您的仓库
5. 构建设置：
   - Build command: `npm run build`
   - Publish directory: `dist`
6. 点击 Deploy site

**优点：**
- 完全免费
- 自动 HTTPS
- 表单处理功能
- 支持自定义域名

---

## 方案二：国内托管（访问速度更快）

### 1. Gitee Pages（码云）

**步骤：**
1. 注册 Gitee 账号：https://gitee.com
2. 创建仓库并上传代码
3. 进入仓库 → 服务 → Gitee Pages
4. 选择部署分支和目录
5. 点击启动

**注意：**
- 需要实名认证
- 免费版有流量限制
- 国内访问速度快

---

### 2. 腾讯云静态网站托管（推荐国内用户）

**步骤：**
1. **注册腾讯云账号**：https://cloud.tencent.com
2. **进入对象存储 COS 控制台**：
   - 搜索 "对象存储 COS"
   - 点击进入控制台
3. **创建存储桶**：
   - 点击 "创建存储桶"
   - 填写存储桶名称（如 `mikun-vision`）
   - 选择所属地域（推荐：北京、上海、广州）
   - 访问权限选择 "公有读私有写"
   - 点击 "下一步" 完成创建
4. **开启静态网站功能**：
   - 进入存储桶 → 基础配置 → 静态网站
   - 点击 "编辑"
   - 开启 "静态网站"
   - 索引文档设置为 `index.html`
   - 错误文档设置为 `index.html`（支持 SPA 路由）
   - 点击 "保存"
5. **上传文件**：
   - 进入存储桶 → 文件列表
   - 点击 "上传文件"
   - 选择 `dist` 文件夹中的所有文件（包括子文件夹）
   - 点击 "确定" 开始上传
6. **获取访问链接**：
   - 进入存储桶 → 基础配置 → 静态网站
   - 复制 "访问域名"（如 `mikun-vision-1251234567.cos-website.ap-beijing.myqcloud.com`）

**优点：**
- 国内访问速度快
- 支持 HTTPS
- 提供 CDN 加速
- 免费额度充足（50GB 存储 + 10GB/月流量）

**费用说明：**
- 超出免费额度后，按实际使用收费
- 通常个人网站每月费用低于 10 元

## 方案三：传统虚拟主机（付费但稳定）

### 推荐服务商：

1. **阿里云虚拟主机**（约 50-100元/年）
2. **腾讯云云服务器**（约 100元/年起）
3. **西部数码**（约 50元/年）

**部署步骤：**
1. 购买虚拟主机
2. 通过 FTP 上传 `dist` 文件夹中的所有文件
3. 绑定域名（可选）
4. 访问网站

- `index.html`

# 双击运行 start.bat
# 访问 http://localhost:3000 确认正常

## 🔧 自定义域名（可选）

### 购买域名：
- 阿里云：https://wanwang.aliyun.com

- GoDaddy：https://www.godaddy.com

### 配置步骤：
1. 购买域名
2. 在托管平台添加自定义域名
3. 在域名服务商处添加 DNS 解析记录
4. 等待 DNS 生效（通常 10-30 分钟）
