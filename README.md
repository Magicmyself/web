## 🚀 快速开始

### 第一次使用
1. 双击 `build.bat` - 同步图片并构建网站
2. 双击 `start.bat` - 启动服务器
3. 打开浏览器访问 `http://localhost:3000`

### 日常使用
- **修改内容后**: 双击 `build.bat`，然后刷新浏览器
- **启动服务器**: 双击 `start.bat`
- **停止服务器**: 在命令行窗口按 `Ctrl+C`

## 📁 文件说明

### 核心文件
- `build.bat` - 更新网站（同步图片+构建）
- `start.bat` - 启动服务器
- `使用指南.md` - 详细使用说明

### 配置文件
- `assets/config/` - 所有文字配置（JSON格式）
- `assets/images/` - 所有图片文件

## 💡 使用技巧

### 添加新图片
1. 将图片复制到 `assets/images/` 对应文件夹
2. 双击 `build.bat`
3. 刷新浏览器

### 修改文字内容
1. 打开 `assets/config/` 中的对应JSON文件
2. 修改内容（保持JSON格式正确）
3. 双击 `build.bat`
4. 刷新浏览器

### 常见问题

**Q: 图片不显示？**
A: 双击 `build.bat` 同步图片

**Q: 服务器无法启动？**
A: 关闭其他占用3000端口的程序，重新双击 `start.bat`

**Q: 构建失败？**
A: 检查配置文件JSON格式是否正确
